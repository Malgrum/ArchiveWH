# Warhammer Club Website

## Description
This website allows users to create army lists for Warhammer 40k and Age of Sigmar (AoS), write battle reports, and includes a codex section for administrators to manage content.

## Features
- User registration and login
- Create and manage army lists
- Write and share battle reports
- Codex section for administrators

## Database
Using MySQL for data storage.

## Project Structure
- **components/**: Contains reusable UI components (buttons, forms, etc.)
- **pages/**: Contains the main pages of the site (home, army lists, battle reports, etc.)

## Installation
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Set up the MySQL database.
4. Run the application using `npm run dev`.
